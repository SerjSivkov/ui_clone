# Changelog — UI Clone

## Unreleased

## 1.0.1

- Feat: экран «О приложении» — версия, лицензия MIT, условия эксплуатации,
  GitHub, кнопка «Пожертвовать» (CloudTips).
- Feat: редактор системного промпта в настройках — шаблоны Flutter /
  React Native / Figma / общий UX, плейсхолдеры {{app}}/{{package}}/{{count}}.
- Feat: структурированный ответ vision (JSON: палитра HEX, экраны,
  компоненты) + markdown; вкладки на экране результата.
- Feat: выбор vision-провайдера — OpenAI, Anthropic, Gemini,
  OpenAI-compatible (свой Base URL); для Anthropic Base URL тоже
  редактируемый (прокси).
- Feat: сжатие JPEG перед upload — качество и max side в настройках.
- Fix: панель управления (оверлей) скрывается на ~150 мс перед каждым кадром,
  чтобы не попадала в MediaProjection-скриншоты.
- Fix: плейсхолдер `{{app}}` — имя приложения из выбранного target или из
  foreground при сборе без выбора; больше не подставляется `unknown app`.
- Fix: ручной «+ кадр» — оверлей снимается с WindowManager на время кадра;
  ImageReader больше не опустошается впустую (на статичном экране кадр не
  приходил и счётчик не рос).
- Feat: прогресс анализа — сжатие N/M, загрузка, ожидание модели, ETA и
  кнопка «Отменить анализ» (CancelToken).
- Fix: вкладка Markdown больше не показывает сырой JSON ответа модели —
  извлекается поле `markdown` / пересборка из palette·screens·components.

## 1.0.0

- Feat: захват экрана (MediaProjection), оверлей и уведомление «Стоп»
- Feat: выбор установленного приложения, AI / офлайн-промпт для клонирования UI
- Feat: умный захват — пропуск почти идентичных кадров (16×16 luma / MAD%),
  порог в настройках, счётчик «дублей пропущено» в UI и уведомлении.
- Feat: режим захвата «по кнопке» — ручной снимок из оверлея (+ кадр) и
  уведомления; режимы таймер / вручную / оба.
- Feat: пауза / возобновление сбора — оверлей («Пауза» / «Далее»),
  уведомление и кнопка на экране сбора; таймер останавливается, «+ кадр»
  на паузе доступен.
- Feat: лимит длительности сессии (1–15 мин, по умолчанию 5) — предупреждение
  за 30 с, автостоп; пауза не расходует лимит; countdown в UI и уведомлении.
- Feat: пропуск кадров, пока UI Clone в foreground — баннер «откройте цель»,
  статус в уведомлении.
- Feat: UsageStats / Accessibility — фильтр кадров по выбранному target
  package, подсказка «сейчас открыто X»; без доступа кадры с target не
  сохраняются (fail closed).
- Feat: компактный оверлей — иконки Пауза/Далее/Стоп/+кадр, перетаскивание
  панели (ручка слева), позиция запоминается.
- Feat: после Стоп (оверлей/уведомление) или автостопа по таймеру приложение
  UI Clone возвращается на передний план.
