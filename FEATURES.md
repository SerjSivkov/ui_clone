# UI Clone — возможности

## Поток обзора UI

- Старт с главного экрана: **Начать обзор интерфейса**.
- Выбор установленного приложения (иконка, label, package) или сбор
  без привязки к app.
- Запрос MediaProjection (системный диалог «Запись экрана»).
- Опциональный оверлей «поверх других окон» с иконками **+ кадр**,
  **Пауза/Далее** и **Стоп**, счётчиком и перетаскиванием панели.
- Foreground-уведомление с действиями **+ кадр**, **Пауза/Далее**, **Стоп**.
- JPEG-скриншоты: режимы **таймер** / **вручную** / **оба** (настройки);
  ручной «+ кадр» в оверлее и уведомлении; умный пропуск дублей.
- Лимит сессии (настраиваемый, по умолчанию 5 мин): предупреждение за 30 с,
  затем автостоп; пауза не тратит лимит.
- Кадры UI Clone не сохраняются, пока приложение само на переднем плане.
- При выбранном target app кадры сохраняются только когда это приложение
  на переднем плане (Accessibility, запасной — UsageStats); подсказка
  «сейчас открыто …». Без доступа фильтр fail-closed.
- Интервал таймера настраивается (по умолчанию 1.5 с), максимум 40 кадров
  за сессию.
- Автозапуск целевого приложения после старта захвата.
- Экран прогресса в UI Clone с превью кадров и кнопкой остановки.

## AI-промпт

- Отправка выборки скриншотов в OpenAI-совместимый `/chat/completions`
  с vision-моделью.
- Промпт описывает: палитру, типографику, навигацию, layout экранов,
  кнопки/поля, функции, рекомендации для Flutter.
- Fallback без ключа / при ошибке сети: структурированный шаблон промпта.
- Копирование в буфер и share.

## Настройки

- API key (secure storage).
- Base URL и имя модели.
- Интервал скриншотов.
- Быстрый переход к разрешению оверлея, Accessibility и статистике.

## Платформа

| Возможность              | Android | iOS      |
|--------------------------|---------|----------|
| Список приложений        | да      | нет*     |
| MediaProjection / кадры  | да      | нет*     |
| Уведомление + Стоп       | да      | —        |
| Оверлей Стоп             | да      | —        |
| Vision / шаблон промпта  | да      | да**     |

\* На iOS нет эквивалента MediaProjection для чужих приложений.  
\*\* UI и офлайн-шаблон доступны; захват экрана чужого app — только Android.
