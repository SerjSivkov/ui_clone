Условия эксплуатации UI Clone

1. Назначение
UI Clone предназначен для анализа интерфейса приложений на вашем устройстве
и генерации текстового промпта (описания стиля, layout и функций) для
последующего воссоздания UX-паттернов. Приложение не является средством
незаконного копирования чужих продуктов.

2. Захват экрана
Для работы на Android используется системный механизм MediaProjection
(запись / снимок экрана). Вы сами выбираете целевое приложение и момент
остановки сбора. Кадры хранятся локально на устройстве в рамках сессии
и используются для анализа.

3. Разрешения
Приложению могут потребоваться: запись экрана, уведомления, показ поверх
других окон (оверлей), доступ к списку приложений, служба специальных
возможностей и/или статистика использования — только для фильтрации кадров
по выбранному приложению. Вы можете отозвать разрешения в настройках
системы.

4. Данные и AI
При указании API-ключа скриншоты (или их выборка) могут отправляться на
указанный вами OpenAI-совместимый endpoint для vision-анализа. Без ключа
приложение формирует офлайн-шаблон промпта локально. Ответственность за
выбор провайдера, ключа и обработку данных несёте вы.

5. Интеллектуальная собственность
Не копируйте чужие товарные знаки, логотипы, охраняемый дизайн и
пользовательский контент в коммерческих продуктах без законных оснований.
Промпт ориентирован на воспроизведение UX-паттернов и визуального языка,
а не на пиратство чужих приложений.

6. Ограничение ответственности
Приложение предоставляется «как есть». Автор не гарантирует точность
анализа, совместимость со всеми устройствами и прошивками и не несёт
ответственности за убытки, связанные с использованием UI Clone, нарушением
прав третьих лиц или утечкой данных через выбранный вами AI-сервис.

7. Пожертвования
Кнопка «Пожертвовать» ведёт на сторонний платёжный сервис CloudTips.
Платежи обрабатываются провайдером; автор не хранит данные вашей карты.

Используя UI Clone, вы подтверждаете, что ознакомились с этими условиями
и принимаете их.
